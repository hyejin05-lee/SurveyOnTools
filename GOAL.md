# 목표

우리의 목표는 유명 Coding Agents : Claude Code, Codex CLI, Antigravity, Cline 그리고 최근 많이 사용되고 있는 특수 목적의 Agents : OpenClaw, oh-my-openagents, Hermes Agent - 들이 제공하는 기능들에 대한 비교표를 만드는 것입니다.

가장 간단한 기능들은 모두 무시해도 됩니다.
- Agentic Memory 를 통한 과거 사용자의 행동에 대한 분석 및 활용
- Harness Engineering
- 사용자와 함께 성장하는 Adaptive Feature (개인화)
- 타 도구와의 연결의 편의성
- Token 절감을 위한 Context Compaction

등 각 도구들이 제공하는 기능들을 표로 정리해서
- 지원하는 기능의 목록
- 각 도구에서 해당 기능의 지원 여부 및 레벨
- 향후 유사한 도구를 개발할 시의 아이디어를 도출하고자 합니다.
- 당장 관심사는 Cline SDK 기반으로 Coding Assistant 개발 시 커버되는 것, SDK 로도 커버가 안되는 기능들을 파악하고자 한다. https://github.com/cline/cline/tree/main/sdk
